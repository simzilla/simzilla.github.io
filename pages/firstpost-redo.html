The First Post (or is it?)

Two First Posts?

What a learning experience this has been. I set out to create this blog in order to learn a little bit about basic web development. I am pretty familiar with Python so I thought that this might be an easy endeavor. What I found is that Python is the least of my worries. There are many production tools that I am not familiar with. Luckily I have little aversion to doing some hard work...accidentally erasing it all and then doing it again.

So Here We Are!

The purpose of this blog is to write about my computer based endeavors and maybe a few other things. I will primarily be writing about my efforts in the realm of web development. Once I get this thing down I'll do a lessons learned blog about creating a blog in Pelican.

Till next time...stay frosty!
